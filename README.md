# Job-Tracker-App
A user interface with authentication to manage your job applications.

## Table of contents
- [Dependencies](#dependencies)
- [Installation](#installation)
- [Setup](#setup)
- [Features](#features)

## Dependencies
### Frontend (client)
- React.js
- Bootstrap
### Backend (server)
- Node.js
- Express.js
- MongoDB via Mongoose Object Data Modeling (ODM) library.
- JSON web tokens (JWT).
- bcrypt for hashing.
